Unordered Lists in HTML

An unordered list is a list in which the list items are not in any specific order. Instead of numbers, unordered lists typically use bullets (•) to mark each item. Unordered lists are created using the <ul> tag, and each list item is defined using the <li> tag.

Basic Example

This is a basic unordered list:

  • Item A
  • Item B
  • Item C

The list above uses the default bullet style to mark each item.

Customizing List Bullet Style

By default, unordered lists use a bullet (•), but we can change the bullet style using the style attribute:

The below example uses square bullets (▪) instead of the default round ones.

  • Item A
  • Item B
  • Item C

The below example uses the default bullets (•).

  • Item X
  • Item Y
  • Item Z

The below example uses circle bullets (◦).

  • Item I
  • Item II
  • Item III

Nested Unordered Lists

Unordered lists can also be nested inside each other:

  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Broccoli
    • Spinach

The nested lists show how unordered lists can be used to represent categories and their respective items.

Unordered lists are perfect for displaying items where the order doesn't matter, such as bullet points or categories. We can customize the list's appearance by changing the bullet style, and also nest unordered lists to create hierarchical structures.
