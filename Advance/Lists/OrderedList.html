Ordered Lists in HTML

An ordered list is a list in which the list items are arranged in a specific order. Ordered lists are created using the <ol> tag, and each list item is defined using the <li> tag.

Basic Example

This is a basic ordered list:

  1. First item
  2. Second item
  3. Third item

The list above is automatically numbered from 1 to 3.

Customizing List Start Number

By default, the list starts from 1. However, we can change the starting number using the start attribute:

  1. Item 5
  2. Item 6
  3. Item 7

In this example, the list starts at 5 instead of 1.

Nested Ordered Lists

We can also nest ordered lists within each other:

  1. Item 1
    1. Sub-item 1.1
    2. Sub-item 1.2
  2. Item 2
  3. Item 3

The list above shows how ordered lists can be nested inside each other.

Note: The nested list continues with new numbering.

Changing Numbering Style

By default, ordered lists use numbers (1, 2, 3). We can change the numbering style using the type attribute:

  1. Item 1
  2. Item 2
  3. Item 3

This example uses uppercase letters (A, B, C) instead of numbers.

  1. Item 1
  2. Item 2
  3. Item 3

This example uses lowercase Roman numerals (i, ii, iii).

Ordered lists are useful when the order of the list items matters. We can easily customize the starting number, the numbering style, and even create nested ordered lists.
