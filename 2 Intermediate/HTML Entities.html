HTML Entities

HTML entities are used to display special characters that are reserved in HTML and also for dsiplaying invisible character like non-breaking spaces in HTML. Also used for displaying characters that are difficult to type with keyboard.

Common HTML Entities

  • & - Represents the ampersand symbol (&)
  • < - Represents the less than symbol (<)
  • > - Represents the greater than symbol (>)
  • " - Represents double quotes ("")
  • ' - Represents single quote (')
  •   - Represents non-breaking spaces
