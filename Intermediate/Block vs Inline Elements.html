Block-level Element: <div>

A <div> element is a block-level element, meaning it takes up the full width available and starts on a new line.

Notice how the <div> elements start on a new line, and the content inside them is separated from other content.

Inline-level Element: <span>

This is an example of an inline element: <span> tag. The span does not create a new line, takes only width that much is necessary.

Inline elements like span do not disrupt the flow of the surrounding text, and they are typically used for styling specific sections or parts of a line of text.
